# 📊 B站UP主数据分析工具

一个专业的B站UP主数据分析工具，帮助UP主通过数据驱动的方式优化内容策略，提升粉丝增长效率。

## ✨ 核心功能

### 🎯 已实现功能

#### **功能B1：视频质量评分系统**
- 对每个视频进行0-100分的综合质量评估
- 多维度评分：播放吸引力、内容质量、传播力
- 自动识别"遗珠"视频（高质量但低曝光）
- 提供具体的改进建议

#### **功能B2：内容选择决策系统**
- 分析各类型内容的表现（游戏、唱歌、模型展示等）
- 推荐最佳内容方向和视频时长
- 分析标题特征的影响（【】标签、表情符号等）
- 识别内容空白区和潜力组合
- 生成TOP N内容推荐

### 📈 生成的图表

**质量评分系统（共3张图）：**
1. 质量评分分布直方图
2. 质量分 vs 播放量散点图（遗珠识别）
3. TOP3视频质量维度雷达图

**内容决策系统（共5张图）：**
1. 内容类型表现对比（分组柱状图）
2. 内容类型分布饼图
3. 视频时长 vs 表现散点图
4. 内容ROI排行榜
5. 标题特征影响对比

## 🚀 快速开始

### 1. 环境要求

```bash
Python >= 3.7
```

### 2. 安装依赖

```bash
pip install pandas numpy matplotlib seaborn openpyxl
```

### 3. 运行分析

#### **完整分析模式**（推荐）

```bash
python run_analysis.py
```

这将执行：
- ✅ 数据加载和清洗
- ✅ 特征工程
- ✅ 质量评分分析（文字报告 + 图表）
- ✅ 内容决策分析（文字报告 + 图表）
- ✅ 生成所有可视化图表

#### **快速分析模式**

```bash
python run_analysis.py --quick
```

只输出核心结论，不生成详细图表（适合快速查看）

## 🔧 配置说明

主要配置在 `config.py` 中：

```python
# 爆款视频定义阈值
VIRAL_VIDEO_THRESHOLD = {
    'play_multiplier': 2.0,      # 播放量 > 平均值的2倍
    'min_play_count': 1000,      # 或播放量 > 1000
    'min_engagement_rate': 5.0   # 且互动率 > 5%
}

# 视频质量评分权重
QUALITY_SCORE_WEIGHTS = {
    'play_score': 0.30,      # 播放吸引力权重 30%
    'engagement_score': 0.40,  # 内容质量权重 40%
    'spread_score': 0.20,    # 传播力权重 20%
    'duration_score': 0.10   # 持久性权重 10%
}
```

可根据实际需求调整这些参数。

## 🎓 使用建议

### 对于新UP主（粉丝 < 1000）
1. 重点关注**内容质量评分**，而不是播放量
2. 优先做"高质量分"的内容类型
3. 关注**遗珠视频**，考虑二次推广
4. 保持稳定的发布频率

### 对于成长期UP主（粉丝 1000-10000）
1. 使用**内容决策系统**找到最佳内容方向
2. 分析**爆款视频特征**，复制成功经验
3. 优化视频时长和标题
4. 尝试不同内容组合，找到差异化定位

### 对于成熟UP主（粉丝 > 10000）
1. 关注**长期运营健康度**（未来功能）
2. 定期分析**内容疲劳度**
3. 探索新的内容类型，避免审美疲劳
4. 建立内容矩阵，多元化发展

## 💡 核心设计理念

1. **高度模块化**：每个功能都是独立的类，便于维护和扩展
2. **配置驱动**：所有参数都在 `config.py` 中集中管理
3. **数据流清晰**：数据加载 → 特征工程 → 指标计算 → 分析报告
4. **代码规范**：详细注释、类型提示、文档字符串
5. **易于扩展**：添加新功能只需继承基类或添加新模块

## 📝 常见问题

### Q1: 中文显示为方框怎么办？
**A:** 在 `config.py` 中的 `CHINESE_FONTS` 列表添加你系统中的中文字体名称。

### Q2: 如何修改质量评分的权重？
**A:** 修改 `config.py` 中的 `QUALITY_SCORE_WEIGHTS` 字典。

### Q3: 如何自定义内容类型？
**A:** 修改 `config.py` 中的 `CONTENT_TYPE_KEYWORDS` 字典，添加关键词规则。

### Q4: 图表太多，只想看某一类怎么办？
**A:** 使用快速模式 `python run_analysis.py --quick`，或修改 `run_analysis.py` 注释掉不需要的部分。
